A HVAC Diagnosis App

Project Overview

This is an application designed for a leading HVAC company, which facilitates the diagnosis and testing of HVAC equipment, thereby enabling technicians to carry out their job efficiently. Worked in conjunctuion with a team of 6 developers, 1 QA, 1 designer and 2 product managers at EY Design Studio.

Technical Stack

  • React Native
  • Redux Toolkit
  • Typescript
  • GraphQL
  • Detox e2e
  • Jest

Project Involvement

  1. Implemented UI designs using Figma and Sketch, translating them into code and front-end logic.
  2. Researched best methods for line and bar charts using a JavaScript charting library for specific data visualization needs.
  3. Researched techniques to optimize code and improve loading time, as well as enhance user interactions.
  4. Collaborated with developers to review and refine pull requests.
  5. Implemented key features for HVAC diagnoses React Native app.
  6. Developed a screen panel that retrieves data from an API related to a specific HVAC product.
  7. Conducted end-to-end testing using Detox and Jest.
  8. Quickly identified and debugged errors within the app.
