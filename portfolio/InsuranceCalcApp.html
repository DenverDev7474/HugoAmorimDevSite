Insurance Risk Assessment App

Project Overview

This is an application that utilizes JavaScript functions to compute health indicator measurements for a risk assessment insurance. Worked in conjunctuion with a team of 2 developers, 1 QA, and 1 product managers at HannoverRe for Robert Half Contract.

Technical Stack

  • React JS
  • HTML, CSS, Javascript
  • AWS Lambdas
  • S3 buckets

Project Involvement

  1. Created JavaScript functions that perform calculations based on user health inputs.
  2. Designed a mobile-responsive interface using bootstrap and custom CSS.
  3. Incorporated language translation features using Vue.js.
  4. Migrated assessment calculators from server-based legacy code to standalone HTML documents
  5. Wrote and tested React components that called files from an S3 Bucket.
  6. Used AWS Amplify to Authenticate Users.
  7. Built routing within the app for specific components.
  8. Coded responsive design based on React-Bootstrap Modules.
