A Medical Platform App

Project Overview

This is a React/Redux application that enables doctors to access patients' health data and determine the appropriate treatments to recommend. Worked in conjunctuion with a team of 4 developers, 1 QA, 1 designer, 1 ux researcher and 1 product managers at EY Design Studio.

Technical Stack

  • React JS
  • Redux Toolkit
  • Typescript

Project Involvement

  1. Researched best methods for line and bar charts using a JavaScript charting library for specific data visualization needs.
  2. Used large sets of data with graphing library to showcase health indicator trends overtime.
  3. Researched techniques to optimize code and improve loading time, as well as enhance user interactions.
  4. Created line and bar charts using a JavaScript charting library, customizing legends, colors, and tick marks.
  5. Quickly identified and debugged errors within the app.
